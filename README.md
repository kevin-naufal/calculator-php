# Kalkulator Web - PHP

Kalkulator online sederhana untuk operasi dasar matematika (+, -, ×, ÷).

## Fitur

- ✅ Form input dua angka
- ✅ Pilih operasi (Tambah, Kurang, Kali, Bagi)
- ✅ Tampilkan hasil di halaman yang sama
- ✅ Validasi input dan penanganan error
- ✅ UI modern dan responsive

## Cara Menggunakan

### 1. Menggunakan PHP Built-in Server

```bash
php -S localhost:8000
```

Kemudian buka browser dan akses: `http://localhost:8000`

### 2. Menggunakan XAMPP/WAMP

1. Salin folder `calculator-php` ke folder `htdocs` (XAMPP) atau `www` (WAMP)
2. Buka browser dan akses: `http://localhost/calculator-php`

## Struktur File

- `index.php` - File utama yang berisi form kalkulator dan logika perhitungan

## Teknologi

- PHP (Server-side processing)
- HTML5
- CSS3 (Modern styling dengan gradient dan animations)
